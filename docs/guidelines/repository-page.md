# Repository Page Guidelines

## File location

docs/repositories/<repository-name>.md

---

## Required structure

```md
---
tags:
  - tag1
  - tag2
---

# Repository Name

## Summary
Short description

## Purpose
Why it exists

## GitHub
https://github.com/andrii2g/<repo>

## Status
Active

## Tech stack
- item

```

## Optional sections
- Features
- Screenshots
- Next ideas

## Writing rules
- short paragraphs
-bullet points preferred
- avoid long text blocks
- be consistent across all repos

---
